# To Buy

A Bun + SQLite + React + Panda starter for saving links to things you want to buy.

## Run it

```bash
bun install
bun run codegen
bun run dev
```

The app expects:

- Bun API server on `http://localhost:3000`
- Vite client on `http://localhost:5173`

## Useful scripts

```bash
bun run dev
bun run build
bun run check
```
